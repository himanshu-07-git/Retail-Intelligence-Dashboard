# 📊 Retail Intelligence Dashboard

![Dashboard Preview](Retail%20Intelligence%20Dashboard.png)

---

## 📌 Overview
This project presents an interactive **Retail Intelligence Dashboard** built to analyze sales performance, customer behavior, and business trends. It helps in extracting actionable insights for better decision-making.

---

## 🎯 Objectives
- Compare sales and orders trends  
- Identify highest sales-performing months  
- Analyze customer demographics (Age & Gender)  
- Evaluate order status distribution  
- Identify top-performing states and categories  
- Analyze sales contribution by different channels  

---

## 🛠️ Tools & Technologies
- Power BI / Excel  
- Data Cleaning & Transformation  
- Data Visualization  

---

## 📊 Dashboard Features
- **Sales vs Orders Analysis** (monthly trends)  
- **Customer Demographics: Age & Gender**  
- **Sales Status Distribution**  
- **Top 5 States by Sales**  
- **Order Channel Analysis (Amazon, Flipkart, Myntra, etc.)**  
- **Interactive Filters (Month, Category, Channel, etc.)**  

---

## 📈 Key Insights
- Women contribute more to total purchases compared to men  
- Certain months show peak sales and order volume  
- Amazon and Myntra are major contributing channels  
- A few states dominate overall sales performance  
- Delivered orders account for the majority of transactions
